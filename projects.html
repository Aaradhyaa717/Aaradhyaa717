1. Study of Gun Violence in United States

Research question: How do gun ownership, population, median income, and political affiliation affect the number of mass shootings or the number of victims of mass shootings per state?

In this project, we examine the correlation between gun ownership, mass shooting victims, and other socio-economic factors. We developed a multivariate regression model to predict the number of mass shootings and victims across states.

2. A Novel Method for Signal Processing Inspired Stock Prediction

This project seeks to identify patterns between various data sources to predict airline stock values. We developed an algorithm for matching changes across data streams and applied multiple linear regression to test its effectiveness. The method filters data to highlight significant trends and remove noise.

3. RSNA 2024 Lumbar Spine Degenerative Classification (Ongoing)

The aim of this project is to aid in the detection and classification of degenerative spine conditions using lumbar spine MR images. We're developing models to simulate a radiologist's diagnostic capabilities.

3. RoyalFlush: BlackJack Game using OCaml

This project is a simple implementation of the classic card game Blackjack, built using OCaml. The game allows players to compete against the dealer, following standard Blackjack rules. Players can hit, stand, double down, and split pairs, while the dealer follows specific rules for drawing cards..
